# 百词斩助手
本应用是类似于谷歌翻译插件的一款浏览器插件，其支持划词翻译，并且可以与百词斩 APP 协同操作。当前支持的功能如下：
- 划词翻译
- 收藏/取消收藏单词（并实时同步到百词斩 APP 中）
- 搜索单词，显示详情信息
- 查看单词本中的单词
- ...
